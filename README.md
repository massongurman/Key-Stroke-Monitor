# Keystroke Monitor

## Overview
This project is a simple Python-based keystroke monitoring tool created for educational cybersecurity learning. It demonstrates how keyboard input can be captured by software and helps illustrate why keylogging poses security and privacy risks.

## Functionality
The program listens for keyboard events and records key presses, including common special keys, to a local text file. The program stops safely when the ESC key is pressed.

## Technologies
- Python  
- Keyboard event listener  
- File I/O  

## Disclaimer
This tool is intended for educational and defensive learning only. Do not use it on systems without explicit permission.
